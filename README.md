# Carrera de coches usando hilos en JavaFX - 

En este ejercicio es una **carrera de coches** con interfaz gráfica donde cada coche corre en un **hilo** independiente. Empleando la clase `Thread` y los métodos `start()` y `run()`,
aparte para este ejercicio usaremos synchronized para nuestro método `finish()` en nuestro **AppController** para proteger el estado de la carrera cuando vayamos a actualizarlo cunado cruecen la meta.
